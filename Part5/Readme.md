# 第５問 - Part5

## 問題内容
１～２６の整数（n）が入力されます。  
その数だけ半角大文字のアルファベットをスペース区切りで出力してください。  
また、規則的な並び（昇順、降順）や、重複は許されません。  
（実行する度に出力が変動する必要があります。）

## 入力条件
```
1 <= n <= 26
```

## 入出力形式

### 入力形式
```
n
```

### 出力形式
```
alphabet_1 alphabet_2 ... alphabet_n
```

## 入出力例

### 例 1
入力例
```
5
```

出力例
```
A H N Z S
```

### 例 2
入力例
```
26
```

出力例
```
I C F E R O J T K V G Z N L X S H M U P Y W Q B A D
```
