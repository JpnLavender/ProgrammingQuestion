# 第３問 - Part3

## 問題内容
１行目に整数で購入する商品の個数（n）が入力されます。  
２行目にスペース区切りで購入する商品の値段（p_n）が入力されます。  
合計の金額に消費税（8%）を加算し出力してください。なお、小数点以下は切り捨ててください。  

## 入力条件
```
1 <= n <= 100
1 <= p <= 100000
```

## 入出力形式

### 入力形式
ここに入力される形式を記述します。
```
n
p_1 p_2 ... p_n
```

### 出力形式
```
sum
```

## 入出力例

### 例 1
入力例
```
3
150 310 50
```

出力例
```
550
```

### 例 2
入力例
```
8
1126 165 429 859 521 20 223 7365
```

出力例
```
11564
```
