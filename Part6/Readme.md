# 第６問 - Part6

## 問題内容
１行目に文字列(s)、２行目に文字(c)が入力されます。  
s に c が含まれていた場合には true、 含まれていなかった場合には false と出力してください。  

## 入力条件
```
0 < s <= 10000
c == 1
```

## 入出力形式

### 入力形式
```
s
c
```

### 出力形式
```
contains
```

## 入出力例

### 例 1
入力例
```
programming
a
```

出力例
```
true
```

### 例 2
入力例
```
github
o
```

出力例
```
false
```
